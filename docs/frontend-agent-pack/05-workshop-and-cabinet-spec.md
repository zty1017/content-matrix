# 05. 资产陈列柜与魔方工坊规格

## 1. 资产陈列柜定位

资产陈列柜不是普通列表，而是：

```text
用户沉淀的内容魔方资产展示空间
+
当前任务自动增强来源
+
魔方工坊资产选择入口
```

视觉：浅色冷调玻璃琉璃手办柜 / 陈列柜。

## 2. 陈列柜状态

三种状态：

- 收起：只露出玻璃边栏，显示资产柜图标、当前相关数量、最近保存提示。
- 半展开：显示当前相关和最近生成两层，系统自动增强和保存入柜时临时使用。
- 全展开：完整上下分层陈列柜，支持搜索、筛选、多选、拖拽、资产预览。

全展开时中央主魔方缩小并略微右移，陈列柜不覆盖主魔方。

## 3. 分层策略

默认智能推荐：

```text
第一层：当前相关
第二层：最近生成
第三层：常用资产
第四层：待确认
第五层：按类型 / 场景 / 时间切换后的资产
```

支持智能推荐、类型、场景、时间、待确认筛选。

## 4. 小魔方展示

每个资产小魔方：

- 默认主重构卡片面朝外；
- 顶面 / 侧面露出来源和类型标签；
- 有独立发光底座；
- 有简短铭牌；
- 悬停轻微旋转；
- 点击进入中央资产预览；
- 拖拽进入当前任务复用或工坊候选。

## 5. 搜索与筛选

陈列柜顶部轻量玻璃工具条：

```text
[搜索资产标题 / 标签 / 来源] [智能推荐] [类型] [场景] [时间] [待确认]
```

搜索结果仍然是陈列柜层板。无结果时显示空展位玻璃层板和温和提示。

## 6. 资产预览模式

```text
资产魔方从展位浮出
→ 中央区域切换为资产预览模式
→ 当前任务魔方半透明留在背景
→ 小型任务锚点出现
→ 右侧显示资产详情
→ 左侧陈列柜保持展开
```

## 7. 复用到当前任务

采用：默认抽取推荐内容块，同时允许用户选择复用哪些部分。

可选择：摘要、证据、偏好、约束、标签、完整卡片。

确认后内容块进入当前任务主魔方核心，沿内部光路分配到对应语义面。

## 8. 关系预览

默认不显示资产连线。只有点击“预览关系”或多选后才显示。

默认只显示当前选中资产的一阶关系，提供“展开更多”查看二阶或全局关系。

关系组可加入工坊。

## 9. 进入魔方工坊

支持拖拽和多选托盘。

托盘：

```text
已选择 3 个魔方
[缩略魔方 A] [缩略魔方 B] [缩略魔方 C]
[清空] [预览关系] [进入魔方工坊]
```

## 10. 魔方工坊定位

魔方工坊是独立沉浸视图，不塞在当前魔方主页面。

布局：

```text
顶部：返回 / 工坊标题 / 选择数量 / 保存草稿
左侧：已选资产魔方队列
中央：拆解重组台
右侧：重组目标与结果预览
底部：重组策略 / 用户输入目标 / 开始重构
```

## 11. 工坊流程

```text
进入工坊
→ 先自动拆解展示可用内容块
→ 用户填写或选择重组目标
→ 系统预选内容块
→ 用户可取消 / 补选 / 调整
→ 新魔方生成
→ 结果态
→ 用户确认保存入柜
```

## 12. 内容块排列

按语义类型归类，并围绕中央新魔方胚胎排列：

```text
          [证据块 Evidence]
                 ↑
[偏好约束] ← [新魔方胚胎] → [摘要观点]
                 ↓
          [标签/场景/来源]
```

语义分区：摘要 / 观点、证据 / 原始线索、偏好 / 约束、标签 / 场景、推理 / 待确认、可复用结构 / 脚本钩子。

## 13. 重组目标

一句话自然语言目标 + 快捷模板按钮：

```text
[知识卡] [决策卡] [经验卡] [流程卡] [选题/脚本] [对比分析]
```

输出风格：

```text
[精简卡片] [决策建议] [知识解释] [脚本复用] [证据严谨]
```

## 14. 工坊结果态

生成后先停留在工坊结果态，不自动保存。

操作：

```text
[继续调整] [重新生成] [保存入柜] [返回资产库]
```

继续调整优先支持：参与内容块、重组目标、输出风格。细粒度字段编辑后续再做。
